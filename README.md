# "Алгоритмы и структуры данных" - Экзаменационное задание 

Вариант 1: Оптимизация маршрутов

## Цель
Разработать систему для логистической компании, которая находит оптимальные маршруты между городами по трём критериям: длина, время, стоимость. Система должна выбирать компромиссный маршрут на основе заданных приоритетов.

## Алгоритмы и структуры данных
- **Граф** — представление сети дорог.
- **Алгоритм Дейкстры** — поиск кратчайшего пути по каждому из трёх критериев.
- **Сравнение маршрутов по приоритетам** — выбор компромиссного решения.

## Входные данные
Файл `input.txt` содержит три секции:
- `[CITIES]` — список городов с ID и названиями.
- `[ROADS]` — двусторонние дороги с параметрами: длина, время, стоимость.
- `[REQUESTS]` — запросы на поиск маршрута с указанием приоритетов.

## Выходные данные
Файл `output.txt` содержит для каждого запроса:
- Три оптимальных маршрута (по длине, времени, стоимости).
- Компромиссный маршрут, выбранный по приоритетам.

## Реализация
Написано на Python 3.13.2. Код соответствует стандартам PEP8, содержит комментарии и обработку исключительных ситуаций.
